# TYDesignableLabel (Swift Package Manager Distribution)

This repository is a **repackaged distribution** of the original `TYDesignableLabel` library to support **Swift Package Manager (SPM)** usage.  
No functional changes have been made to the original source code.

---

## üì¶ Purpose

- The original repository did not provide SPM support.
- This repository restructures the original code so it can be easily integrated into modern iOS/macOS projects using Swift Package Manager.
- All original logic and implementation remain intact.
- This repository contains a **Swift-converted** version of the original `TYDesignableLabel` library, along with structural adjustments to support **Swift Package Manager (SPM)**.
